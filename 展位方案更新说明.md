## 🛠 技术实现

### PDF处理流程
1. **提取**: 使用 PyMuPDF (fitz) 从PDF导出45页效果图
2. **分析**: 根据页码位置推断展位面积 (18/27/36㎡)
3. **生成**: Python脚本自动生成495行HTML展位卡片代码
4. **集成**: 动态加载到主页面，支持筛选和预览

## 🎯 使用指南

### 客户端操作
1. **打开页面**: 访问 https://raysu1008.github.io/AIGreenExhibition/
2. **选择尺寸**: 在"展位规格"下拉框选择 18/27/36㎡
3. **自动筛选**: 展位库自动显示对应尺寸的展位
4. **查看详情**: 点击展位卡片打开详细信息弹窗
5. **预览图片**: 点击展位图片放大查看高清效果

### 管理员维护
- **添加展位**: 修改 `loadBoothTemplates()` 函数中的 `booths` 数组
- **更新图片**: 上传到 `展位方案集/` 文件夹并更新GitHub
- **调整样式**: 修改 CSS 中的 `.template-card` 样式
- **修改筛选**: 调整 `filterTemplates()` 函数逻辑
